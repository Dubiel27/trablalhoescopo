Modo Escuro (Simulação)

De Wikipédia, a enciclopédia livre.

Tópico de Demonstração

O modo escuro (em inglês: *dark mode*) é um esquema de cores de interface de usuário que usa um fundo escuro com cores de texto claras. Este design é frequentemente discutido em termos de design de interface do usuário e web design.

  • **Lançamento:** 2018
  • **Uso:** Reduzir cansaço visual

O **modo escuro** é um tema de cores que exibe texto claro (como branco ou cinza) em um fundo escuro (como preto ou cinza escuro). O principal objetivo é reduzir a tensão ocular causada por telas brilhantes em ambientes de pouca luz.

Em julho de 2024, a Wikipédia implementou oficialmente o modo escuro para usuários logados. Essa mudança reflete uma tendência geral na tecnologia, onde muitos sistemas operacionais e aplicativos oferecem essa preferência ao usuário.

Introdução ao Escuro

Historically, o uso de texto claro em fundo escuro remonta às primeiras telas de computador CRT, que exibiam fósforo verde ou âmbar em um fundo preto. Esse estilo, no entanto, foi invertido com o advento da impressão a laser e das interfaces gráficas de usuário (GUIs), que buscavam imitar "papel" branco.

Vantagens

As vantagens do modo escuro são frequentemente citadas tanto em ergonomia quanto em eficiência energética. No entanto, o consenso sobre os benefícios visuais nem sempre é unânime, dependendo da condição ocular do usuário.

Cansaço Visual

Muitos usuários relatam que o tema escuro reduz o cansaço visual, especialmente durante o uso noturno. A redução da quantidade de luz emitida pela tela é crucial para este benefício.

Economia de Bateria

Em telas OLED e AMOLED, o modo escuro é conhecido por economizar significativamente a energia da bateria, pois os pixels pretos ficam completamente desativados.

Implementação (CSS)

A forma mais moderna de implementar o modo escuro em CSS é usando a *media query* @media (prefers-color-scheme: dark), que detecta a preferência do sistema operacional do usuário. Alternativamente, pode-se usar variáveis CSS para facilitar a troca de tema, como neste exemplo:

/* Exemplo de uso de variáveis CSS */
:root {
    --bg-conteudo: #2c2c2e;
}
body {
    background-color: var(--bg-conteudo);
}
